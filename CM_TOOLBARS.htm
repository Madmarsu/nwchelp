Toolbars Command

This command is used to enable or disable the listed toolbars. You can use this command to hide or restore toolbars that are available to speed your access to common commands. The toolbar names that are checked are those that will be made visible. Those that are not checked will be removed from the screen.

If you have a piano control included in your toolbar configuration, you can also select the piano style that should be used.

Piano Style

The piano styles are defined by a piano.ini file, which is generally found in the program files folder, or in the Config Files section of Tools, Options, Folders.
