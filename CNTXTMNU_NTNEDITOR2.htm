Editor Context Menu Commands (Selection Present)

The following commands are available in the notation editor when one or more notation items are currently selected in the active staff. These can be accessed either by: hitting the right mouse button (on a two or three button mouse), or by pressing the Menu key on the computer keyboard. After this action, the appropriate notation editor context menu pops up with the choices below:

  • Cut: Copy the selection to the clipboard and then delete it
  • Copy: Copy the selection to the clipboard
  • Clear: Clear the selection (no effect on clipboard)
  • Paste: Paste the contents of the clipboard into the staff
  • Last User Tool: Used to repeat the most recently executed user tool on the selected notation
  • User Tool: Used to run your own custom tools/commands on selected notation
  • Notes:
    • Beam: Beams the note flags of the selected notes
    • Triplet: Turns the selected notes into a triplet
    • Crescendo: Toggles the crescendo attribute of the selected notes
    • Diminuendo: Toggles the decrescendo attribute of the selected notes
    • Accent: Toggles the accent attribute of the selected notes
    • Dot: Marks subsequent notes and rests as having a dotted duration
    • Double-Dot: Toggles the double dot attribute
    • Slur: Slurs the selected notes
    • Staccato: Toggles the staccato attribute of the selected notes
    • Tenuto: Toggles the tenuto attribute
    • Tie: Marks subsequent notes as being tied to the next note
    • Stem Up: Toggles the upward stem attribute
    • Stem Down: Toggles the downward stem attribute
  • Filtered Properties: Allows an include/exclude filter to be applied before editing properties
  • Properties: Open the propeties editor for the current selection
