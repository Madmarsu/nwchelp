Customize Toolbars

This window is used to customize the command toolbars available in the program. Using this window, you can:

  • Add a new toolbar
  • Remove an existing toolbar
  • Change the name of an existing toolbar
  • Add new command to an existing toolbar
  • Rearrange the commands included in a toolbar
  • Reset the toolbars to the factory default

All changes to the toolbars are saved in the file nwc2toolbars.dat, in the Config folder. The Config folder must be accessible for creating and saving new files in order for the customizations to be saved for future sessions.
