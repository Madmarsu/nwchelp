Note Command

This command adds an audible music note to the active staff. It places the note at the insertion point using the note parameters that are set in the Notes menu. The current view of the score is automatically reformatted to align all notes in the proper positions. This command can be quickly executed by pressing the <Enter> key on the computer keyboard while in the score editor, or by selecting the Note command on the editor's context menu.
