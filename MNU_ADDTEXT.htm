Text Command

This command is used to add text instructions to the staff. These instructions do not influence the automated performance of the staff.

Display Font: You can indicate a specific font that should be used when displaying the text. These are defined in File, Page Setup.

Scaling: You can use the scaling option to control the actual size of the text, relative to the the default size specified in the font setup.

Char Map: Opens a character grid for selecting specific glyphs from the current display font.
