Audit Accidentals

This command can be used to remove any unnecessary accidentals assigned to notes in the active staff. It compares all notes with the running key signature, and if the accidental it contains matches the running signature, then the accidental is removed.

Note that this audit does not currently support the automatic placement of hint accidentals that many composers use to remind the player of the implied accidental associated with a note that is in key, but had been altered in the prior measure.

This audit will not detect if the accidental is the enharmonic version of a note that appears in the key signature. If you want all note spelling to be converted to in key variants, you can use the Audit Enharmonic Spelling command.
