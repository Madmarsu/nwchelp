Cycle Noteheads Command

This command cycles through the available noteheads during note entry. This enables all of the notehead styles to be selected in a rotating fashion using the Q key on the computer keyboard. This command only works when entering new notes. Changing the notehead of selected notes that are already on the staff involves selecting the specific notehead that should be applied to the selection.

The notehead styles that are cycled using this command include:

Standard notehead based on duration
X notehead
Diamond notehead
Blank notehead
