Cycle Noteheads Command

This command cycles through the toolbar notehead buttons during note entry. This enables assorted notehead styles to be selected in a rotating fashion using the Q key on the computer keyboard.

The notehead styles that can be cycled using this command if they are found in the toolbars include:

Standard notehead based on duration
Non-pitched notehead
Diamond notehead
Blank notehead
X notehead
‘do’ Note (see Shape Notes)
‘re’ Note
‘mi’ Note
‘fa’ Note
‘fa’ Note (Alt)
‘so’ Note
‘la’ Note
‘ti’ Note
