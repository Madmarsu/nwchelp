Goto Command

This command is used to move the insertion point to a specified measure. The measure must be greater than the starting bar number associated with the score.

This command is also useful on occasions where you want to find out the current measure number at the insertion point. By selecting this command, you can find out the current bar number because it is the initial value presented in the bar number field. You can simply cancel the operation if you are only wanting to determine the current measure number.

You can create or extend a selection by enabling the Selection checkbox.
