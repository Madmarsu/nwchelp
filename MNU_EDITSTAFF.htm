Staff Properties

This command is used to change the properties of the current staff. A series of property tabs are used to represent all of the parameters that you can change about the staff. The following property tabs are associated with staff data:

  • General: This includes the name, group and ending bar style for the staff.
  • Visual: This includes options that control the height, color, and overall style of the staff.
  • Midi: This includes many of the play back options that control the performance of the staff through your sound card or speakers.
  • Instrument: This tab allows you to set the default instrument that should be used during play back of this staff.
