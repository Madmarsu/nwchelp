Export Command

This command is used to export a work into an alternative file format, such as a Standard MIDI File (*.mid), NoteWorthy Composer text file, or the older NoteWorthy Composer version 1.75 file format. You can select:

  • Type 0 MIDI File: contain a single one track that contains all of the song data
  • Type 1 MIDI File: Each staff reqpresents a MIDI track in the file, as well as a conductor track for performance tempo and other signature data
  • NoteWorthy Composer 1.75 File: A representation of the score that is suitable for opening in NoteWorthy Composer 1.75 or later
  • NWC Text File: A textual representation of the score that is suitable for reading in a text editor, and processing by text filter/conversion programs
