Export Command

This command is used to export a work into a Standard MIDI File (*.mid) or the older NoteWorthy Composer file format (*.nwc).

  • Type 0 MIDI File:
    A performance file with a single track that contains all of the song performance data.

  • Type 1 MIDI File:
    A performance file where each staff represents a separate MIDI track. A conductor track contains performance tempo and other performance/signature instructions.

  • NoteWorthy Composer 1.75 File:
    Creates a file that can be opened by NoteWorthy Composer 1.75. Any version 2 specific functionality in the file will be lost in the conversion.
