Import Command

This command imports an existing music file that is not in NWC song format.

You can import the following file types:

  • MID Files
    These files are assumed to be Standard MIDI Files. These are commonly used to exchange music performance data between music sequencer, notation, and algorithmic composition software. When imported, the performance data is converted into standard NWC music notation.
