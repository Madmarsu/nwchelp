New Command

This command is used to create a new NoteWorthy Composer composition. A new file is started by selecting a song template and then pressing OK. The template named <Blank Score> in the General tab is a special template that starts a new file from scratch.

You can create your own templates by dragging and dropping an existing NWC song file onto the current list of templates. A copy of the file will be made and placed into the same folder as the other templates in the list. The list of templates is generated from the song files contained in the Template folder inside the NWC application folder. Any first level sub-directories in the Template folder are placed into their own tabs in the template list.

Once a new file has been started, you can edit the new song as any other song being edited in the notation editor. The first time you chose to save the new file, the Save As dialog (see Save As Command) will automatically appear so that you can specify the filename for this new song.

More about templates

The templates feature uses standard *.nwc song files as the basis for song templates. A template can be any standard nwc file. The difference between using a template and opening an existing file is that a new file is created without an association to a pre-existing file. When a new file is created based on a template, the template name is only used during the construction of the new file. Later attempts to save the file will need to be named with a new filename.

Customizing

All templates are files contained in the Template sub-folder of the NWC application directory. The Template folder can contain one additional level of sub-folders that will be recognized by the template feature. Each sub-folder underneath the Template folder represents a new tab in the File New dialog. You can create your own tabs in the File New dialog by adding folders inside the Template folder.

New templates can be added to the system in three different ways:

  1. A song file can be saved into one of the Template folders in the NWC application directory. This is as easy as a Save As command on the File menu.
  2. A file can be copied into the Template folder using the explorer or file manager programs.
  3. A file can be dragged from the explorer and dropped inside a template list in the File New dialog. This will automatically make a copy of the file and place the copy in the appropriate Template folder.
