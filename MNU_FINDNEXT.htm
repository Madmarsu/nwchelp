Find Next Command

This command is used to locate the next item in the staff that matches criteria established by a previous Find operation. This can be used to locate a series of matching elements in the staff very quickly, without having to scroll around browsing for the elements. The quickest way to initiate successive searches using the same criteria is to first use Ctrl+F to establish the Find criteria, then use F3 to repeat the find using the same criteria.
