Note Colors

This command is used to select the color highlight that should be used for the notehead of the next added note, or the item color for other objects. When the default color is selected, the object or notehead will use the default color for the staff or item.

If one or more items are selected on the staff, then the color buttons always change the item color for the selected items. Notehead colors can be changed as a group using Edit, Properties.
