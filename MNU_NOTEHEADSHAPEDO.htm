Shape Note Commands

These commands select various shape notes for use during note entry. When applied to a note, the notehead will change to the selected shape. This can be selected prior to inserting the note, or applied to existing notes on the staff by selecting them and then applying this command. Different noteheads are added in a chord by first selecting the appropriate notehead, then adding the note to the chord in the customary fashion.

The shape note commands include the following tool buttons:

‘do’ Note
‘re’ Note
‘mi’ Note
‘fa’ Note
‘fa’ Note (Alt)
‘so’ Note
‘la’ Note
‘ti’ Note
