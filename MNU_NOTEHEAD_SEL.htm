Notehead Selector

This is a drop-down selector for notehead styles. It can also be opened from the keyboard using the Q key.

The notehead styles that can be cycled using this command include:

Standard notehead based on duration
Non-pitched notehead
Diamond notehead
Blank notehead
X notehead
‘do’ Note (see Shape Notes)
‘re’ Note
‘mi’ Note
‘fa’ Note
‘fa’ Note (Alt)
‘so’ Note
‘la’ Note
‘ti’ Note
