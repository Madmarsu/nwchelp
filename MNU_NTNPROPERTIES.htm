Notation Properties Command

This command is automatically executed when the Edit | Properties command is selected for one or more selected items on the staff. A series of property tabs matching the selected items is shown on the screen. These tabs give you the opportunity to change one or more attributes of the selected items.

One or more of the following property tabs could be shown, depending on the selection:

  • Clef Properties
  • Key Signature Properties
  • Time Signature Properties
  • Bar Line Properties
  • Special Ending Properties
  • Tempo Properties
  • Tempo Variance Properties
  • Performance Style Properties
  • Flow Direction Properties
  • Dynamic Properties
  • Dynamic Variance Properties
  • Sustain Pedal Properties
  • Instrument Patch Properties
  • Multi-point Controller Properties
  • Text Expression Properties
  • Expression Placement Properties
  • Note Properties
  • Rest Properties
  • Visibility Properties
  • Statistics (future use)
