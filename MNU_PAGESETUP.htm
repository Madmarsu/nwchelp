Page Setup Command

This command allows you to set properties associated with a printed page of your song. The page options are distributed across several tabs:

Contents: One or more staff parts can be selected for editing and/or printing using this tab:

  • Groups
  • Visible Parts

Fonts: This is where you can customize the fonts used in the staff. The page includes the following controls:

  • Notation Typeface
  • Print Size
  • Font List
  • Modify Button

Options: This is where the general setup options are available. They include the following (click on the field name to get additional information):

  • Page Numbering From
  • Title Page Info
  • Extend last system
  • Increase note spacing for larger note durations
  • Staff Labels
  • Measure Numbers
  • Measure Start

Margins: This is where the page margins are set. The margin settings include the following (click on the field name to get additional information):

  • Top of Page
  • Inside Edge
  • Outside Edge
  • Bottom of Page
  • Mirror Margins
  • Measure Units

Preview: This is where you can preview the effects of changes made in the other page setup areas. This area includes a summary list of what the current view of the score will look like on the printed page. You can also go directly to print preview from this page. The page includes the following:

  • Information List
  • Preview Button
