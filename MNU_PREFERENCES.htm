Options Command

This command enables program customization. Once initiated, a dialog is presented with the following tabs:

  • Color: This is where the highlight colors can be customized
  • Editor: This is where the editor can be customized
  • File: This is where the File handling of NoteWorthy Composer is customized
  • Folders: This is where the file location handling of NoteWorthy Composer 2 is customized
  • Import: This is where the MIDI import options are customized
  • Midi: This is where MIDI play back behavior is customized
  • Record: This is where MIDI input and recording featured are customized

Any changes you make in a tab will take effect when you press OK, or switch to another tab. Once you switch to another tab after having made changes, you can no longer cancel the changes that were made in the prior tab.
