Score Review

This command brings up a summary list of all staves in the score. The list can be sorted by any column by clicking on the header label. There are many more columns available than are initially shown. You can right click on any column header to add additional columns, or to hide the field in the current column.

Column fields include:

Basic Fields
ID This is the index order for the staff list. It shows the position of each staff in the score.
Name The is the physical name of the staff.
Warn This shows the number of warnings that have been found for the staff. Currently, this only indicates the number of unassigned octave accidentals found in the staff.
Hide This reflects whether the staff is currently shown in the editor and when printing.
General
Label This is the full staff label assigned to the staff.
Label Abbrev This shows the abbreviated staff label if one is assigned to the staff.
Group Shows the staff group assigned from Staff Properties.
Performance 
Muted Indicates if the staff is muted.
Port This is the "0 based" MIDI port assigned to the staff.
Device This is the actual MIDI device name assigned to the staff.
Channel This is the "1 based" MIDI channel for the staff.
Patch This is the "1 based" instrument patch assigned to the staff from Staff Properties (note that it differs from the 0 based value that is assigned there). A value of 0 here represents that a patch change is not being sent by this staff.
Transpose This is the transposition assigned to the staff from Staff Properties.
Volume This is the initial MIDI volume assigned to the staff from Staff Properties.
Stereo Pan This is the initial MIDI stereo pan position assigned to the staff from Staff Properties.
Stats/Metrics
Bars This is the number of bar lines found in the staff that are preceded by one or more notes. This count includes bar lines that have the "Exclude From Bar Count" option enabled. In most cases, this number should match amongst all visible staff parts.
Play Span This is the total duration/length of all notes and rests found in the staff when they are summed together during a simulated performance. In most cases, this number should match amongst all visible staff parts.
View Span This is the total duration/length of all notes and rests found in the staff when they are all individually summed together, ignoring any repeats or flow directions. In most cases, this number should match amongst all visible staff parts.
