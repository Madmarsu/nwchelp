Manage Objects

This command brings up a list of all currently installed object types, as well as any new object types found in the current file.

You can use the Online Synch tool to automatically compare all of your object types with their online versions. If an online version exists and it differs from your current version, then you can double click the object type to update it with the online version. This is the recommended way of updating your plugins when newer versions are published by the plugin author.

You can browse the online library of available plugins from our web site.
