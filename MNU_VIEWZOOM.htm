Zoom

The size of the notation font used in the NoteWorthy Composer 2 score editor can be specified with this command. The size that you specify is the number of pixels which separate each line in the staff. Note that only an even number of pixels can be applied to staff separation. If you enter an odd number, it will be converted to the next even value (based on whether you are increasing or decreasing the separation).
