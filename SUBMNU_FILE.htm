File Menu Commands

The File menu contains the necessary commands to open, save, and print NoteWorthy Composer files. All NoteWorthy Composer notation files use a file type extension of nwc. The naming convention is handled automatically by the NoteWorthy Composer 2 file control commands. Help on the File manipulation commands is provided below:

  • New: Creates a new score file from scratch or using a predefined template
  • Open: Opens an existing score file or MIDI performance
  • Open Backup: Open a backup version of a *.nwc file
  • Open Sample: Quick way to open a sample score file provided with the program
  • Save: Saves the score in the active edit window
  • Save As: Saves the score in the active window to a new file name and/or file type
  • Save All: Saves all open score files which have been changed
  • Revert: Reverts the currently active score back to the version saved on disk.
  • Close: Closes the currently active score
  • Import: Import a music file
  • Export: Export to a different file format
  • Info: Specifies authoring information about the active score
  • Page Setup: Sets page display options when printing
  • Printer Setup: Changes the current printer configuration
  • Print Preview: Previews what will actually print
  • Print: Prints the active score
  • History: Scores which have been recently open
  • Exit: Quits the program
