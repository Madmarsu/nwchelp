Notes Menu Commands

This menu is where the running parameters associated with note entry are maintained. In order to expedite note, chord, and rest entry, all note parameters are controlled through this menu. Each new note or rest that is added to the staff utilizes the parameters that are checked in this menu.

  • Color: Select one of the highlight colors for the next notehead or item
  • Durations
    • Whole: Checked prior to adding a whole note/rest
    • Half: Checked prior to adding a half note/rest
    • Quarter: Checked prior to adding a quarter note/rest
    • Eighth: Checked prior to adding a eighth note/rest
    • Sixteenth: Checked prior to adding a sixteenth note/rest
    • Thirty-second: Checked prior to adding a thirty-second note/rest
    • Sixty-fourth: Checked prior to adding a sixty-fourth note/rest
  • Noteheads
    • Standard Notehead: Select standard noteheads that are based on note duration
    • X Notehead: Selects an X notehead for use in note entry
    • Harmonic Notehead: Selects an open diamond harmonic notehead for use in note entry
    • Blank Space Notehead: Selects a blank notehead for use in note entry
    • Non-pitched Notehead: Selects a non-pitched notehead for use in note entry
    • Diamond Notehead: Selects a diamond notehead for use in note entry
    • Notehead Shape Notes: A variety of shape notes can be selected for use
  • Accidentals
    • Natural: Future notes will include a natural sign
    • Flat: Future notes will include a flat sign
    • Sharp: Future notes will include a sharp sign
    • Double Sharp: Future notes will include a double-sharp sign
    • Double Flat: Future notes will include a double-flat sign
    • Courtesy Accidental: Accidentals will include a courtesy indication
  • Crescendo: Toggles the crescendo attribute of the selected notes
  • Diminuendo: Toggles the decrescendo attribute of the selected notes
  • Accent: Toggles the accent attribute of the selected notes
  • Dot: Toggles the dotted note attribute
  • Double-Dot: Toggles the double dot attribute
  • Grace: Toggles the grace note attribute of selected notes
  • Marcato: Toggles the marcato attribute of the selected notes
  • Slur: Toggles the outbound slur attribute of the selected notes
  • Staccato: Toggles the staccato attribute of the selected notes
  • Staccatissimo: Toggles the staccatissimo attribute of the selected notes
  • Tenuto: Toggles the tenuto attribute
  • Tie: Toggles the outward note tie attribute
  • Stem Up: Toggles the upward stem attribute
  • Stem Down: Toggles the downward stem attribute
  • Persistent Tools: Configures which tools should persistent after a new item is placed into the staff
