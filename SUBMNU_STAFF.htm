Staff Menu Commands

This menu allows staff parts to be added, changed, or removed from the score.

  • New Staff: Add a new staff to the current view of the score
  • Import Recording: Create a new staff by importing the most recent recorded MIDI performance
  • Move Staff: Move the currently active staff to another position in the score
  • Delete Staff: Remove the currently active staff
  • Mute Staff: Mute the active staff so that it does not sound during play back
  • Staff Properties: Edit the properties of the current staff
