Tools Menu Commands

This menu contains various tools for working with a score file.

  • Score Review: Review the score in a list format
  • Layerng Tool: A layering tool for quickly reworking staff layers
  • Mute List: Set and clear the mute status for the score parts from a list
  • Audit
    • Audit Accidentals: Audits and polishes the accidental assignments in the active staff
    • Audit Bar Lines: Audits and replaces the bar line structure of the current staff
    • Audit Enharmonic Spelling: Audits and re-spells notes to match the current key signature
    • Audit Note Stem: Audits the note stem directions for selected notes
  • Play
    • Record: Records MIDI input while playing the active score
    • Staff Play: Plays just the active staff
    • Play: Plays the active score
    • Pause: Pause the current play back
    • Stop: Stops play or resets all configured devices
    • Insert MIDI Input: Enables remote note entry via MIDI Input
    • Windows Volume Control: Opens the system volume control panel
  • Automatic Beam: Automatically beaming function for the active staff
  • Transpose Staff: Transposes the active staff up or down in semitones
  • Force Accidentals: Forces all notes in active staff to be labeled with an accidental
  • User Tool: Used to run your own custom tools/commands on selected notation
  • Manage Objects: Used to manage object types and matching plugin source
  • Options: Configure NoteWorthy Composer 2 application preferences
