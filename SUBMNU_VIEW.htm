View Menu Commands

This menu contains various commands that control appearance and zoom levels in the editor and surrounding toolbars.

  • Show Active Staff Edges: Show the top and bottom boundary edges for the currently active staff
  • Show Crosshair Command: Show crosshair guidelines at the insertion point
  • Show Label Panel: Show or hide the entire staff label panel on the left side of the editor
  • Show Staff Labels: Show staff labels in the left side panel of the editor
  • Show Bar Number Panel: Show or hide the bar number panel at the top of the editor
  • Layering in Edit Mode: Enable/disable staff layering while in edit mode
  • Viewer Mode: Places the editor in Viewer mode, where hidden and editor only elements are not shown
  • Zoom In: Zoom into the currently active score
  • Zoom Out: Zoom out of the currently active score
  • Zoom: Specify a specific zoom level for the active score
  • Refresh Score: Redraw (refresh) the active score
  • Status Bar: Toggle the visual status of the status bar
  • Tool Bars: Select which tool bars should be currently visible
