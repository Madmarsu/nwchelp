Clef Properties

Clef Type

Chose the type of clef that you want to use from the drop-down list. Treble, bass, alto, tenor, and percussion clefs are all supported from within this box.

Octave Shift

You can apply an octave shift to the clef using this box. You can elect to have notes played an octave up or down from the standard pitch associated with the selected clef.

Disable Automatic System Clef

This option can be used to prevent this clef from automatically being displayed at the start of later systems in the printed copy of the file. This option is needed even when the first clef instance is hidden from the printed copy via Visibility settings.
