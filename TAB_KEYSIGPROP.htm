Key Signature Properties

A key signature can specified either by selecting its name from the predefined signature list, or by manually selecting the sharps and flats and creating a User Defined key signature. The specific option controls include:

  • Predefined Key Signature List

    In most cases, you will want to use a standard key signature from this list. Select User Defined if you want to enter the sharps and/or flats yourself manually in the definition area below.

  • User Defined Key Signature

    This area shows the sharps and/or flats that are included in the key signature. If you have selected User Defined from the list above, you can enter your own key signature here.

  • Hide Cancellations

    A new key signature will generally show natural signs where necessary to cancel out the prior key signature. This option is used to prevent the display of the natural signs.

  • Disable Automatic System Key

    This option can be used to prevent this key change from automatically being displayed at the start of later systems in the printed copy of the file. This option is needed even when the first key change instance is hidden from the printed copy via Visibility settings.
