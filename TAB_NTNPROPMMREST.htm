Multi-measure Rest Properties

The multi-measure rest object is used to quickly add multiple bars of rest onto the staff. The number of bars of rest can be set from this tab, and then the following options control how the bars of rest will appear on the printed page.

When Visible on Printed Page

  • Show Only Once

    You can use this option to force the multi-measure rest to display the total number of bars of rest in a multiple staff printout, even if it did not fit entirely within the first system. When this is set, the MMR itself will not appear in later systems, but whole bar rests or empty bars might appear in its place, depending on the other options.

When Hidden

  • Print Bar Lines

    When the MMR is hidden, this option will cause bar line boundaries to be displayed on the printed page for each empty measure of rest.

  • Print Whole Rests

    When the MMR is hidden, this option will cause a whole rest to be displayed on the printed page for each measure of rest.
