Custom Object Properties

This is the general property page for items added via the Insert Object Command. All custom objects have the following built-in properties:

  • Object Type
    This defines the type of object, which links the object to its external plugin script.

  • Pos
    All objects have a vertical position on the staff. In the editor, a tear drop anchor is shown for every object at this position on the staff.

  • Class
    The object class can be used to enable a special behavior. When an object is assigned to the StaffSig class, it will be automatically included in every printed page system, in a similar fashion as is done with Clef and Key Signature items.
