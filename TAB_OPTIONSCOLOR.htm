Color Options Dialog Tab

This tab allows the editor background color to be set, as well as a variety of colors used within the score. Color can be set for the following notation elements:

  • Printer Default: Any notational elements that do not explicitly set a color will use this color when printed.
  • Active Staff: This is the color that will be used for the active staff in the editor, unless the staff color has been changed in Staff Properties.
  • Inactive Staff: Any staff that is not active in the editor will use this color, unless the staff color has been changed in Staff Properties.
  • Hidden Notation: Notation that will not appear in print will be shown with this color while in the editor.
  • Play Highlight: Notes will be highlighted in this color as they are played.
  • Highlight Colors: Miscellaneous colors that can be assigned to staff elements via their property interface.
