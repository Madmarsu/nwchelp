Color Options Dialog Tab

This tab allows the editor colors to be set, as well as the color of various notational elements. The highlight colors can be utilized via the staff notation Visibility property.

The following color options are available:

  • Printer Default: Any notational elements that do not explicitly set a color will use this color when printed.
  • Staff Label Panel: This is the background color used for the staff label panel shown on the left side of the editor.
  • Edit Background: This is the primary background color used in the notation editor control.
  • Active Background: This is the background color used in the area that surrounds the currently active staff.
  • Active Staff: This is the color that will be used for the active staff in the editor, unless the staff color has been changed in Staff Properties.
  • Inactive Staff: Any staff that is not active in the editor will use this color, unless the staff color has been changed in Staff Properties.
  • Active Hidden Notation: Notation that will not appear in print will be shown with this color while active in the editor.
  • Inactive Hidden Notation: Notation that will not appear in print will be shown with this color while inactive in the editor.
  • Play Highlight: Notes will be highlighted in this color as they are played.
  • Highlight Colors: Miscellaneous colors that can be assigned to staff elements via their property interface.
