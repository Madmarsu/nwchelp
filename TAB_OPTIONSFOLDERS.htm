Folders Options Dialog Tab

The following options are available for controlling the file location settings in NoteWorthy Composer 2:

  • File Type List

    This list summarizes the default folder locations that can be changed.

  • Modify Button

    Use this button to change the location for one of the folder locations.

You can use the built-in path expansion strings to point to several common folders used by the system.
