Import Options Dialog Tab

The following options are available for controlling the MIDI import operations of NoteWorthy Composer:

Notate Options

  • Notate Pedal Marks
  • Insert Instrument Changes
  • Automatic Beam
  • Detect Compound Time
  • Note Resolution
  • Rest Resolution

Track Splitting Options

  • Separate MIDI channels
  • Maximum single track chord size
