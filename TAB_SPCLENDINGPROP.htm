Special Ending Properties

Special ending properties include:

  • Standard Special Endings (1 through 7)

    Using these boxes, you can set the different endings that apply to the section that follows this item in the staff. Up to seven standard endings are supported. You can select more than one ending in a single item. These are used in conjunction with master repeat open and close bar lines to build a repeated section.

  • Default (D.S./D.C.) Ending

    This is the default ending for a repeated section. The default is used after a Dal Segno or Da Capo have been executed, since master repeats are themselves ignored after one of the flow directions have been executed.

  • Continue prior ending

    You can use this button to add a continuation line that extends through the current bar.

  • Closed ending

    This option is used to terminate the special ending bracket just before the next bar.

  • Hide ending text

    This option can be used to hide any ending text while in Viewer mode or when printing. If necessary, custom text can be added using Insert, Text Expression.

  • Bracket height

    This controls the height of the starting and closing ends of the ending bracket.

  • Vertical offset

    This controls the vertical position of the ending relative to the current staff upper boundary. A positive offset moves the ending upward away from the top of the staff. A negative offset moves the ending downward. The offset starts at the staff's dynamic upper boundary.
