Text Expression Properties

Text expression properties include:

  • Expression

    Type the text or symbol characters that you want to include in the staff into this box.

  • Display Font

    You can control the look of the entered character string using this box. You can select one of the predfined staff text fonts, a user font, or the current staff notation symbol font.

  • Char Map

    This button allows you to insert a character or symbol into the existing expression. You select the symbol from the character map or grid that is displayed. You can double click a symbol to quickly insert it into the expression. If you are working on a computer that employs a standard multi-byte character system, you might experience difficulties when using this utility.
