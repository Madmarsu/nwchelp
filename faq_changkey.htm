FAQ: Changing the key signature

It is easy to change the key signature of a staff, as follows:

  1. Use Tools: Force Accidentals to assign an accidental to each note.
  2. Select the key signature, and change it using Alt+Enter (Edit Properties). If no key exists, then use Insert: Key Signature (or K)   to enter a key signature at the beginning of the staff.
  3. Select Tools, Audit, Audit Enharmonic Spelling to update notes to the key.
