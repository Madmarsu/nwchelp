FAQ: Step entry using a MIDI keyboard

If you have a MIDI capable keyboard, soundcard, and the necessary cable connections, you can enter notes at your own pace from the keyboard. This is usually referred to as step time entry from a remote keyboard, and can be setup as follows.

  1. Tools: Options, Record tab
  2. If not set, select the Input Device Driver for your system
  3. MIDI tab - set MIDI Port Usage to While Active or Always
  4. Tools: MIDI Input Active

You can now use the MIDI keyboard will enter the pitch of a note or chord,and the pitch bend wheel can be used to set with the note duration prior to entering the pitch.
