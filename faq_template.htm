FAQ: Create your own templates

You can change the templates provided in NoteWorthy Composer 2 and save them as new templates:

  1. Open the rhythm that you want to change
  2. Change the template as you want
  3. Select File | Save As
  4. Save file under Template, Rhythm as whatever name you want
  5. Now you have a new template the way you want it
  6. When you enter Ctrl+N, you template will be there

You can also create a brand new template:

  1. Create your song file
  2. Select File | Save As
  3. Save under Template, Rhythm or Custom
