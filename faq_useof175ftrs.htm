FAQ: Using new 1.75 features

The significant new features added in version 1.75 can be utilized by following the directions included below:

  • Find Command
    The Find command is available on the Edit menu. You can use it to find simple text on the staff, or do more complicated find expressions to search for any object on the staff.
  • Lines Staff Property
    The Lines property can be found in Staff Properties, on the Visual tab. This option allows you to specify the number of lines that a staff shows in the score. You can even set this to zero, so no staff lines will appear.
  • Muted Property for Notes
    The Muted property is found from the Properties command on the Edcit menu when you have one or more notes selected. This allows one or more notes to be muted during play back without having to mute the entire staff.
  • No Leger Lines Property for Notes
    The No Leger Lines property is found from the Properties command on the Edit menu when you have one or more notes selected. This allows you to make leger lines disappear for selected notes. This can occasionally be useful in creating visual effects on the staff.
  • New Standard Lyric Syllable Placement
    Lyric syllables can now be aligned to their notes in one of two ways. They can either be aligned using standard rules, or using the technique found in earlier versions of the program, at the start of accidentals/notes. The new option can be found in the Lyric Configuration tab, under the heading of Align Syllable Using.
