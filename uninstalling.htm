Removing NoteWorthy Composer

NoteWorthy Composer 2 includes an uninstall utility in the product. A selection is added to the remove software section of the Add/Remove Software applet in the control panel. This makes the removal of the application a relatively simple process. It can be done as follows:

  1. Close any and all NoteWorthy Composer 2 application windows.

    Windows will not allow the removal of application files that are currently in use. If you are currently using NoteWorthy Composer 2, you must exit before continuing these steps. You will also need to close this help file or Windows will prevent you from deleting it.

  2. Run the remove utility.

    *** W A R N I N G ***
    This will remove the NoteWorthy Composer 2 application folder. If you have custom files or settings contained in this folder, make sure that you back them up before proceeding.
